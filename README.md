
# LSTM Sentiment Analysis

## Objective
Detect sentiment from text reviews using an LSTM-based model.

## Setup

```bash
pip install -r requirements.txt
```

## Train Model

```bash
python model_training.py
```

## Predict (CLI)

```bash
python predict.py
```

## Web Interface

```bash
streamlit run web_interface.py
```

## Example Output

Input: "I loved the movie!"  
Output: Positive
